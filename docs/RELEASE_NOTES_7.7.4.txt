R+ 7.7.4 (labs repo, consolidar y COAG)
=======================================

Fecha: 2026-07-23

## Resumen

**7.7.4** corrige el import del repositorio cuando hay electrolitos/gases seriados (la BH matutina ya no se copia a todas las tomas), deja que Consolidar arme grupos a mano, y evita que COAG tome el TTP como INR o el mínimo del rango como TP.

## Nuevo / mejorado

- **Repo / pegado masivo** — Historial respeta ventana ≤2 h y no fusiona gasometría con gasometría; la BH de la mañana queda solo en el bloque matutino.
- **Consolidar historial** — Lista los conjuntos del día; tú armas grupos (mismo día) y solo se fusionan esos. Ya no ofrece «fusionar todo el día» por defecto.
- **COAG (TP / TTP / INR)** — Extracción más estricta en PDFs incompletos: no cruza al estudio siguiente ni usa el mínimo del rango cuando falta el resultado.

## Instalación

Descarga desde: https://github.com/mausalas99/r-mas/releases/tag/v7.7.4

- Mac: `R+-7.7.4-arm64.dmg`, `R+-7.7.4-x64.dmg` (y zip para auto-update).
- Windows: `R+-7.7.4-x64.exe`.

Tras el build local: `npm run build:mac` / `npm run build:win` (incluye write-release-yml.js).
