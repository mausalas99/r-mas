R+ 3.4.0 (estable — R+ Móvil + tutorial LiveSync)
================================================

Fecha: 2026-05-17

## Resumen

Sincronización en iPad y teléfono por navegador (misma Wi‑Fi que el Mac anfitrión), sin app aparte. Misma interfaz visual que la app de escritorio; el tutorial de Sala e Interconsulta termina explicando LiveSync y el enlace móvil.

## R+ Móvil (nuevo)

- **URL** `http://<IP-anfitrión>:3738/mobile/` (o `/join?code=…` que redirige ahí) abre **la misma UI R+** en Safari.
- **Oculto en móvil:** Generar Nota, Indicaciones, Listado Word y Salida rápida (.docx).
- **Sincronización en sala LiveSync:** pacientes, laboratorios, pendientes y agenda (misma sala que escritorio).
- **Panel ⇄:** botón **Copiar enlace móvil** además de la invitación completa.

## Tutorial (onboarding)

- Tras terminar el recorrido **Sala** o **Interconsulta**, dos pasos nuevos: **LiveSync en escritorio** (⇄, salas) y **versión móvil** (Safari, misma red).

## LAN (heredado de 3.3.x)

- Código por defecto **1234**, merge de expediente en sala, Copiar labs compacto (3.3.1+).

## Quién debería actualizar

- Equipos que consultan la lista en **iPad** durante la ronda.
- Usuarios que ya usan ⇄ y quieren compartir el enlace móvil con el turno.

## Instalación

Descarga desde: https://github.com/mausalas99/r-mas/releases/tag/v3.4.0

- Mac: `R+-3.4.0-arm64.dmg`, `R+-3.4.0-x64.dmg` (y zip para auto-update).
- Windows: `R+-3.4.0-x64.exe`.

Tras el build local: `npm run build:mac` / `npm run build:win` (incluye `write-release-yml.js`).
