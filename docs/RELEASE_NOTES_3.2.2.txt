R+ 3.2.2 (estable — laboratorio + actualizaciones en canal Estable)
====================================================================

Fecha: 2026-05-16

## Resumen

Versión estable recomendada tras **3.2.0** / **3.2.1**. Incluye los arreglos de **Laboratorio** de 3.2.1 y corrige el **auto-updater** en canal **Estable**: detecta releases oficiales (p. ej. desde **3.0.6** o **3.2.0** hacia 3.2.2) sin tener que usar Pre-releases. El distintivo **Pre-release** en el modal solo aparece si GitHub marca la versión como borrador.

## Actualizaciones

- **Canal Estable** — El canal elegido en Ajustes se guarda en disco y se aplica al arrancar; al cambiar entre Estable y Pre-releases se vuelve a buscar en GitHub.
- **Buscar actualizaciones** — Sincroniza el canal antes de consultar; en Estable se usa `/releases/latest` (releases oficiales).
- **Modal** — Ya no muestra “Pre-release” solo por tener activado el canal de borradores; solo si la release en GitHub es pre-release.

## Laboratorio (desde 3.2.1)

- **Copiar** en Resultados visible de nuevo en vista **Normal**.
- **BH extendida apagada** — Primera línea compacta (Hb, Hto, VCM, HCM, Leu, Neu, Eos, Plt); RBC, CHCM, RDW, MPV y Ret en la segunda línea solo con la preferencia encendida.
- **Valores alterados** — Conservan **\*** al copiar o enviar a nota; en pantalla asterisco rojo (sin mezclar “, alterado” al seleccionar).

## Quién debería actualizar

- Usuarios en **3.0.x**, **3.1.x**, **3.2.0** o **3.2.1**: Ajustes → **Estable** → **Buscar actualizaciones…**, o instalar el DMG/EXE de esta versión.
- Si el auto-update falla (p. ej. firma en Mac): descarga manual desde Releases.

## Instalación

Descarga desde: https://github.com/mausalas99/r-mas/releases/tag/v3.2.2

- Mac: `R+-3.2.2-arm64.dmg`, `R+-3.2.2-x64.dmg` (y zip para auto-update).
- Windows: `R+-3.2.2-x64.exe`.

Tras el build local: `node scripts/write-release-yml.js` (o incluido en `npm run build:mac` / `npm run build:win`).
