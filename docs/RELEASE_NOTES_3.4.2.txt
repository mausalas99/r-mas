R+ 3.4.2 (estable — antibiograma R/I/S y copiar condensado)
===========================================================
Fecha: 2026-05-17
## Resumen
Antibiograma en Laboratorio y Cultivos con una sola R, I y S (hover con detalle por antibiótico). «Copiar informe completo» pega el resumen condensado, no el informe crudo de SOME.
## Nuevo / mejorado
- **Resultados (cultivos):** chips R / I / S como en Cultivos, con panel al pasar el cursor (fármaco, CMI, interpretación).
- **Cultivos — Copiar informe completo:** copia fecha + cabecera + línea `ATB R: … | I: … | S: …` (por aislamiento en polimicrobianos).
- **Copiar (Laboratorio):** sigue exportando el texto compacto de R+ (`buildLabLines`).
## Instalación
Descarga desde: https://github.com/mausalas99/r-mas/releases/tag/v3.4.2
- Mac: `R+-3.4.2-arm64.dmg`, `R+-3.4.2-x64.dmg` (y zip para auto-update).
- Windows: `R+-3.4.2-x64.exe`.
Tras el build local: `npm run build:mac` / `npm run build:win` (incluye `write-release-yml.js`).
