# VPO (Valoración preoperatoria) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Segmento **VPO** en expediente (Clínico en Interconsulta, Salida en Sala) con calculadora de riesgo alineada al Excel, machotes EKG/Rx editables, plan farmacológico perioperatorio desde SOME (Medicamentos), persistencia por paciente y copiar al portapapeles para nota externa.

**Architecture:** Núcleo puro en `vpo-calculator.mjs` + catálogos en `vpo-lookups.mjs`; estado en `vpoByPatient`; UI en `vpo-panel.mjs` montada en `#vpo-container`. Misma lógica en ambos modos; `expediente-tabs.mjs` enruta el pane a Clínico o Salida según `appMode`.

**Tech stack:** Vanilla JS (ESM), `node --test` (`*.test.mjs`), `localStorage` vía `storage.js`, UI en `public/partials/layout/app-body.html` + `npm run build:ui`.

**Spec de referencia:** `docs/superpowers/specs/2026-05-29-vpo-design.md`

- [ ] **Step 2: Run — FAIL**

- [ ] **Step 3: Implement `computeVpoScores`**

Reglas:

- **RCRI:** sumar 1 por cada flag true en `rcri` (incl. `cirugiaAltoRiesgo`); `riskLabel` por puntos 0 / 1 / 2 / ≥3 con textos hoja *Estratificación*.
- **ARISCAT:** edad ≤50→0, 51–80→3, >80→16; SpO₂ ≥96→0, 91–95→8, ≤90→24; IR último mes→17; Hb≤10→11; incisión; duración ≤2→0, 2–3→16, >3→23; urgente→8.
- **Caprini:** sumar puntos según flags (definir tabla mínima alineada Excel: edad≥75, IMC>25, varices, reposo, EVC previo, trombofilia, esteroide, artritis — usar puntos del Excel donde estén en inputs R24–R30).
- **Gupta:** `linear = guptaIntercept + asaCoef + functionalCoef + procedureCoef + creatinineClassCoef` (coeficientes en `vpo-lookups`); `micaPercent = 1 / (1 + Math.exp(-linear))`; interpretación `<1%` Bajo, `1–1.9%` intermedio, `≥2%` Alto.
- **Creatinina Gupta:** `<=1.5` coef 0; `>1.5` según lookup si aplica.

Exportar `computeVpoScores(input) → { asaClass, rcri, gupta, ariscat, caprini }`.
