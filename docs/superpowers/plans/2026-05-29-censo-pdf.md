# Censo de guardia (PDF) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Exportar un PDF landscape con todos los pacientes activos en modo Sala, alimentado por expediente (diagnósticos en Datos del paciente, ATB/meds editables, accesos con fecha), reemplazando el censo en Google Drive.

**Architecture:** Estado canónico en `patient` (`diagnosticosList`, `censoMedsText`, `accesoFecha`); agregador puro `censo-build.mjs` arma filas; `censo-export.mjs` muestra diálogo y llama `POST /generate-censo`; `generate-censo.js` dibuja tabla con pdf-lib (sin plantilla AcroForm). Encabezado: mes/fecha automáticos + equipo desde Mi Perfil (`residenteR2`, `residenteR1`, `profesorName`).

**Tech stack:** Vanilla JS (ESM), `node --test` (`*.test.mjs` / `*.test.js`), Express (`server.js`), `pdf-lib`, Electron document pipeline (`requestDocumentJson` / `handleDocumentGenerateResponse`), `npm run bundle:renderer`.

**Spec:** `docs/superpowers/specs/2026-05-29-censo-pdf-design.md`

```javascript
// public/js/patient-diagnosticos.mjs
import {
  parseDiagnosticosText,
  formatDiagnosticosCopy,
} from './vpo-dx-inference.mjs';

export function ensurePatientDiagnosticos(patient) {
  if (!patient) return;
  if (!Array.isArray(patient.diagnosticosList)) patient.diagnosticosList = [];
  if (!patient.diagnosticosList.length && patient.diagnosticosText) {
    patient.diagnosticosList = parseDiagnosticosText(patient.diagnosticosText);
  }
  if (!patient.diagnosticosList.length) patient.diagnosticosList = [''];
  var nonEmpty = patient.diagnosticosList
    .map((d) => String(d || '').trim().toUpperCase())
    .filter(Boolean);
  patient.diagnosticosText = formatDiagnosticosCopy(nonEmpty);
}

export function diagnosticosTextForCenso(list) {
  return (list || [])
    .map((d) => String(d || '').trim().toUpperCase())
    .filter(Boolean)
    .join(' + ');
}

export function migratePatientDiagnosticosFromVpo(patient, vpoState) {
  if (!patient || !vpoState) return false;
  var has = (patient.diagnosticosList || []).some((d) => String(d).trim());
  if (has) return false;
  var from = (vpoState.diagnosticosList || []).filter((d) => String(d).trim());
  if (!from.length) return false;
  patient.diagnosticosList = from.map((d) => String(d).trim().toUpperCase()).concat(['']);
  ensurePatientDiagnosticos(patient);
  return true;
}

export function applyPatientDiagnosticosList(patient, list) {
  patient.diagnosticosList = list;
  ensurePatientDiagnosticos(patient);
}
```
